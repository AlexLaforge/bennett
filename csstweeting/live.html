Welcome to CSS Tweeting

You control how you want this page to look through tweets.

Any CSS you tweet to the account @csstweeting will be applied to this page. Want to make the font all in Comic Sans? Maybe you want to make the background color lime green?

The website uses the Twitter Search API to query tweets sent to @csstweeting. With these tweets wrapped in a style tag, Iâ€™m giving you the keys to style this page however you please. Newer CSS tweets are lower on the stylesheet and override earlier tweets.

This concept is a fork of an similar project I made years ago, called CSS Creatures, which allows users to create a little animated creature by tweeting.

Some of the HTML tags on this page you can style include body, header, footer, h1, h2, h3, p, a, ul, li, code, and img.

Restricted CSS

For a small bit of sanity, some features in CSS are restricted. This includes two things:

  • url()
  • content property

Otherwise, you can make this page as ugly or as pretty as you want. It only takes 15-20 seconds after tweeting for your CSS to be applied to this page.
